第七章

选择器优先级+样式优先级+常用css3

一.选择器优先级

  • 后写的相同选择器会覆盖前面的
<head>

    .box{

    width:500px;

    height:500px;

    background:Red;

}

    .box{

    width:50px;

    height:50px;

    background:green;

}

</head>

<html>

    <div class='box'></div>

</html>
  • 后代选择器的优先级情况
    1. ID选择器 > class选择器 > 标签选择器
    2. ID选择器个数不相等,ID个数多的优先级高
    3. ID选择器个数相等,则看class选择器个数,个数多者优先级高
    4. class个数相等,则看标签选择器个数
    5. 选择器个数相同,顺序不一样,他们的优先级是一样的 
      

      #wrap ul li.list{}

      

      .wrap ul li #list{}

选择器的使用原则就是准确控制选中元素

二.样式优先级

  • 行内样式
  • 内部样式
  • 外部样式
<link rel='stylesheet' type='text/css' href='url.css' />

注意:外部样式需要设置编码 @charset 'UTF-8';
外部样式 = (跟顺序有关)内部样式 < 行内样式
补充:
1. @import url(style.css);样式引入
2. !important 提升优先级(最高)

div{background:red !important;}

三.常用css3样式

  • 圆角的写法

    1. border-radius:0px 0px 0px 0px;
    2. border-radius:0px 0px 0px;
    3. border-radius:0px 0px;
    4. border-radius:0px;

  • 圆角单样式写法

    1. border-top-left-radius
    2. border-top-right-radius
    3. border-bottom-right-radius
    4. border-bottom-left-radius

  • 圆形
    border-radius:50%;

  • 盒子阴影
    box-shadow:0px 0px 0px 0px color;
    1. 第一个值 代表水平偏移量(必需)
    2. 第二个值 代表垂直偏移量(必须)
    3. 第三个值 模糊半径(可选)
    4. 第四个值 阴影大小(可选)
    5. 第五个值 阴影颜色值(可选)
    6.inset 内部

box-shadow:0px 0px 0px 0px color;
  • 超出隐藏
    overflow内容溢出处理方式
    1. visible 默认可见
    2. hidden 超出隐藏
    3. scroll 出现滚动条
    4. auto 根据内容情况来决定是否出现滚动条
    5. 单独给一个轴给超出处理方法 overflow-x/y:;另一个值就是auto
