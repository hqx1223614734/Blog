定位属性position

一.什么是定位

二.默认定位static

  • 浏览器默认位置,元素的正常 位置
    1. 直接设置left/right/bottom/top 无效过
    2. z-index层级也不会有效果

三.相对定位relative

  1. 相对于它本身正常位置发生位置改变
  2. 通过 left/top/right/bottom 属性来改变位置
  3. 特点
    1. 不使元素脱离文档
    2. 不影响元素本身特性
    3. 如果没有设置定位偏移量,对元素本身没有任何影响
    4. 就算元素发生位置改变,但它任然占据着它原来位置的空间
li{

    width:50px;

    height:50px;

    background:red;

    list-style:none;

}

.box{

    position:relative;

    left:100px;

}

<ul>

    <li></li>

    <li class='box'></li>

    <li></li>

</ul>

四.绝对定位absolute

  1. 相对于已经定位的上级元素来发生位置改变
    定位上级包括(参照物):relative,absolute,fixed
  2. 通过 left/top/right/bottom 属性来改变位置
  3. 特点
    1. 完全脱离文档流
    2. 行内元素支持宽高
    3. 块级元素内容撑开宽高
    4. 没有上级定位则相对于body
    5. 配合上一级定位一起使用
    6. margin:auto;无效
.box{

    width:50px;height:50px;border:1px solid red;position:relative;

}

.box-1{

    width:20px;height:20px;background:red;position:absolute;left:50px;

}

<div class='box'>

    <div class='box-1'></div>

</div>

  1. 相对于浏览器窗口定位
  2. 随着内容一起滚动
