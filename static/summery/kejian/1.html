三.HTML超文本标记语言

超文本标记语言,标准通用标记与眼下的一个应用.”超文本”就是指页面内容可以包含图片,链接,音乐,程序等非文字元素.结构包括头部head,主题body.头部提供关于网页信息,主体部分提供网页具体内容.

四.网页主体结构

  • 1. 标记
    <>
  • 2. 最外层标记<html></html>
    也称之为跟标签
  • 3. 文档头<head></head>
    不会显示在浏览器中,基本的描述
  • 4. 网页主体<body></body>
    正文:文字,图片,链接,表单等
  • 单标签
    <img />
    <input type='text'>
  • 双标签
    开始标签+闭合标签

六.超链接

  1. 超链接可以是一个字,一个词,或者一组词,也可以是一幅图,也可以是一些内容,或某部分
  2. 鼠标放在超链接部分时,会变一只小手

  3. 通过a标签使用
    1. 通过href属性 - 创建指向另一个文档的链接
    2. target属性定义网页打开方式
    _blank默认新窗口
    _self规定当前窗口

    <a href="https://www.tmall.com/" target="_self">天猫</a>

  4. 锚点
    1. 通过name名字,可以通过id属性来代替name

    <a href='http://www.w3school.com.cn/'>走你</a>

    <a href='http://www.w3school.com.cn/' target='_blank'>走你</a>

  5. 创建邮件

    <a href='mailto:someone@microsoft.com?subject=Hello%20again'>发送邮件</a>

    <a href='mailto:someone@microsoft.com?cc=someoneelse@microsoft.com&bcc=andsomeoneelse2@microsoft.com&subject=Summer%20Party&body=You%20are%20invited%20to%20a%20big%20summer%20party!'>发送邮件</a>

  6. 下载
    当我们链接的是一个软件的时候
