二. 元素类型

  • 元素两大阵营

  1. 块级元素
    1.独占一行
    2.默认宽度100%.支持宽高.行高
    3.支持外边距和内边距
    4.可以包裹内联元素和其他块元素
    5.p不内包裹块元素
    常见块元素
    div / dl / form / h1-h6 / ol / ul / p

  2. 内联元素
    1.也叫行内元素
    2.和其它内联元素同处一行
    3.不支持宽高
    4.外边距上下无效.auto无效
    5.内边距只对内联元素产生影响.无视块元素
    6.标签之间空格被解析
    7.a能包裹块元素
    常见内联元素
    a / b / em / i / img / input / lable / span
    [ img / input 支持部分块级元素特性的内联元素]

  • 元素的转换

转换为块元素/显示
display:block;
转换为内联元素
display:inline;
行内块元素
display:inline-block;
影藏对象
display:none;

  • 行内块元素
  1. 横排显示
  2. 支持宽高
  3. 没有宽高时内容撑开
  4. 支持margin.但不支持auto
  5. 标签间隔被解析

三.垂直外边距合并

当两个垂直外边距相遇时,他们将形成一个外边距.合并后的外边距的高度等于两个发生合并的外边距中高度最大的那一个.

  • 两个同级元素发生合并
    当一个元素出现在另一个元素上面,第一个元素下外边距与第二个元素上外边距发生合并.
<div class='box1' style='width:500px;height:500px;background:red;'>

<div class='box2' style='width:400px;height:400px;background:green;'>
  • 父子级发生合并

当一个元素包含在另一个元素中时,它们的上/下外边距也会发生合并.

<div class='box1'>

    <div class='box2'></div>

</div>
  • 空元素上下元素合并

一个空元素,它有外边距,但是没有内容和边框,它的上下外边距也会发生合并.

如果这个外边距遇到另一个元素的外边距,它也会发生合并

  • 外边距合并解决办法
  1. 两个同级元素发生合并
    1 给第二个元素浮动
    2 给第二个元素绝对定位
    3 单独个某一个元素设置ineine-block
  2. 父子级发生合并
    1 加边框
    2 给父级设置内边距代替子级外边距
    3 给父级一个overflow:hidden;
    4 设置绝对定位
    5 设置display:inline-block;
