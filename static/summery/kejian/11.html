第十一章

表单 form

一.定义

  1. form表单是一个包含表单的元素区域
  2. 表单元素是允许用户在表单中(文本域.下拉列表.单选框.复选框等)输入信息的元素

二.form详解

<form action='' method='' name='' target=''></form>
  1. action 规定当提交表单时向何处发送表单数据
  2. method 规定用于发送数据的方法get/post
  3. name 名字
  4. target 规定action属性中提交的页面会在何处打开_blank/_self

三.form中的控件 input

<input type='' name='' value=''/>
  1. type中的类型有
    1. text 文本
    2. password 密码
    3. radio 名字 指定相同的name
    4. checkbox 复选框
    1)label 标签为input元素定义标注
<input type='radio' name='' id='a'/>
<label for='a'>**</label>
for属性应当与相关元素的id

2)checked 属性定义默认选中的input元素

<input type='checkbox' name='checked'/>

3)disabled 属性规定应该禁用input元素

<input type='checkbox' name='disabled'/>

5. submit 提交form中的数据
6. reset 重置
7. button 按钮
8. image 图片(有提交功能)
9. hidden 隐藏
10. select/option 下拉选框 (高度的支持不兼容)

<select size='2'></select>
size 规定下拉列表中可见选项的数目
<opation selected='selected'></opation>
selected 规定在select里面默认展示低级行

11. textarea 文本域
1) rows属性定义 高度
2) cols属性定义 宽度
3) resize 调整尺寸样式属性 none/vertical/horizontal
12. fieldset 可将表单内的相关元素分组
13. legend 为fieldset元素定义标题

<fieldset><legend>定义域标题</legend><input type='text'/></fieldset>

14.focus伪类 (适用a,input,button)
1)outline 外轮廓线

input:focus{background:red;}

15. placeholder 属性提示信息

<input type='text' name='' id='' placeholder='请输入密码'/>

四.总结
